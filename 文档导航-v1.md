## 5. 里程碑交付物映射

### M1（认证中心最小可用）

- API：登录、刷新、CLI 审核流
- 数据库：核心表与索引完成
- 会话状态机：`active/suspended_readonly/revoked` 生效

### M2（文本聊天闭环）

- Chat 侧鉴权中间件接入
- `text.send/text.recv` 权限校验稳定
- `json/command/file` 仅后端能力保留，不开放 UI

### M3（管理端可用）

- 用户、角色、权限、设备审核页面可用
- 审计检索与导出可用
- 高危操作二次确认可用

## 6. 变更管理约定

- 需求变更先更新 `统一认证与双Web端需求规格-v1.md`
- 设计变更同步更新详细设计和 API 文档
- 安全策略变更必须同步更新演练清单与 Runbook
- 所有变更需记录版本号与变更日期
